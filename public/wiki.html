02 · Found a bug?

Send us the debug log

FT8AF writes a structured event log every time it runs. Pull it off your phone in two taps, attach it to a GitHub issue, and we can usually pinpoint what went wrong — instead of guessing.

  • Where to find debug.log
  • Share it via Files, Drive, or email
  • What to include in the issue
  • No phone? No problem.
Report a bug

This wiki is intentionally short.

If something here doesn't match what you see in the app, that's a docs bug — open an issue and we'll fix it. For Q&A-style answers (pricing, supported radios, the FT8CN relationship) head over to the FAQ.
